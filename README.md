# yt-dlp-installer-CLI
Install yt-dlp, ffmpeg and a CLI for easy and quick video downloading.

Allow YOUTUBE videos and mp3 downloading by simply entering the link of the video (using yt-dlp).

## Installation process

The programm will :

- Take the .exe files from the github detailed further down

- Put those .exe files in a new file located at the root in \yt-dlp

- Add a path variable to allow the usage of yt-dlp via CLI anywhere

- Create an executable file to launch the client CLI
